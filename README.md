## Installation

```bash
$ npm install
```
## MongoDB
There is a docker-compose.yml file for starting MongoDB database docker container.
```bash
$ docker-compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

$ npm run test

# e2e tests
```
